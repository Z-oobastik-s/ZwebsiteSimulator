# Zoobastiks - Тренажёр Слепой Печати

Современный онлайн тренажёр слепой печати для обучения быстрой и точной печати вслепую.

## 🚀 Возможности

- **Уроки для всех уровней**: Начинающий, Средний, Продвинутый
- **Мультиязычность**: Русская, Английская, Украинская раскладки
- **Тест скорости**: 60-секундный тест на максимальную скорость
- **Мультиплеер**: Соревнуйся с друзьями в реальном времени
- **Система достижений**: Зарабатывай монеты за прохождение уроков
- **Магазин уроков**: Покупай новые интересные уроки за монеты
- **Статистика**: Отслеживай свой прогресс и улучшай результаты
